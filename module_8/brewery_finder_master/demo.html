üß™ Test Scenarios

1. Search by City

  • Select "Search by City"
  • Try: Austin, Denver, Portland
  • Expected: Multiple breweries in each city

2. Search by Name

  • Select "Search by Name"
  • Try: Dogfish, Stone, Lagunitas
  • Expected: Specific breweries matching the name

3. Test Pagination

  • Search for a city with many breweries (e.g., San Diego)
  • Use Previous/Next buttons to navigate
  • Expected: 12 results per page, proper navigation

4. Test Error Handling

  • Try searching with 1 character (should show validation error)
  • Try searching for non-existent cities (should show "No results found")
  • Expected: Clear error messages and guidance

5. Test Search History

  • Perform multiple searches
  • Click on history items to repeat searches
  • Expected: Recent searches saved and clickable

6. Test Responsive Design

  • Resize browser window or use mobile view
  • Test on different screen sizes
  • Expected: Layout adapts to screen size
